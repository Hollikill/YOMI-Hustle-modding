build process for windows:

requirements:
- have cmake installed
- have visual studio 2022 installed

1. create a build folder in this directory
2. run 'cmake ..' in the build directory
3. open the .sln in the build folder
4. Set ColorTests as the startup project
5. click on the 'Local Windows Debugger' button to run

If the program doesn't run after this, try to follow the steps again
